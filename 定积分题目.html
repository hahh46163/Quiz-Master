定积分计算

困难 建议用时: 15分钟

计算定积分：\(\int_{0}^{1} (2x + e^x) dx\)，要求写出详细的积分步骤，并说明所用的积分公式。

题目提示

  • 使用基本积分公式：\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)（\(n \neq -1\)）
  • 指数函数积分公式：\(\int e^x dx = e^x + C\)
  • 定积分性质：\(\int_{a}^{b} [f(x) + g(x)] dx = \int_{a}^{b} f(x) dx + \int_{a}^{b} g(x) dx\)

解答步骤

1
拆分积分
2
计算原函数
3
代入上下限
4
得出结果
1

拆分积分表达式

根据定积分的线性性质，将被积函数拆分为两个部分：

\(\int_{0}^{1} (2x + e^x) dx = \int_{0}^{1} 2x dx + \int_{0}^{1} e^x dx\)

2

分别计算原函数

根据基本积分公式，计算两个部分的原函数：

\(\int 2x dx = x^2 + C\) （使用公式：\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)）

\(\int e^x dx = e^x + C\) （使用指数函数积分公式）

3

代入上下限计算

应用牛顿-莱布尼茨公式 \( \int_{a}^{b} f(x) dx = F(b) - F(a) \)：

\(\int_{0}^{1} 2x dx = \left[ x^2 \right]_{0}^{1} = 1^2 - 0^2 = 1\)

\(\int_{0}^{1} e^x dx = \left[ e^x \right]_{0}^{1} = e^1 - e^0 = e - 1\)

4

计算最终结果

将两部分结果相加，得到定积分的最终值：

\(\int_{0}^{1} (2x + e^x) dx = 1 + (e - 1) = e\)

（其中 \(e \approx 2.71828\)）

解答正确！

常用积分公式

基本积分公式

\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\) （\(n \neq -1\)）

\(\int \frac{1}{x} dx = \ln|x| + C\)

指数函数积分

\(\int e^x dx = e^x + C\)

\(\int a^x dx = \frac{a^x}{\ln a} + C\) （\(a > 0, a \neq 1\)）

三角函数积分

\(\int \sin x dx = -\cos x + C\)

\(\int \cos x dx = \sin x + C\)

\(\int \sec^2 x dx = \tan x + C\)

定积分性质

\(\int_{a}^{b} [k_1 f(x) + k_2 g(x)] dx = k_1 \int_{a}^{b} f(x) dx + k_2 \int_{a}^{b} g(x) dx\)

\(\int_{a}^{b} f(x) dx = -\int_{b}^{a} f(x) dx\)

\(\int_{a}^{b} f(x) dx = \int_{a}^{c} f(x) dx + \int_{c}^{b} f(x) dx\)
